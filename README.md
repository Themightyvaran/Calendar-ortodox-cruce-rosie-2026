# Calendar-ortodox-cruce-rosie-2026

## Descriere (Română)
Calendar Ortodox 2026 – Sărbători cu Cruce Roșie.  
Acest proiect conține un fișier `.ics` compatibil cu Google Calendar, Outlook, Apple Calendar și Samsung Calendar.  
Include doar sărbătorile ortodoxe românești cu cruce roșie pentru anul 2026.

### Import
Poți descărca fișierul și importa în aplicația ta de calendar preferată.

---

## Description (English)
Orthodox Calendar 2026 – Red Cross Holidays.  
This project contains an `.ics` file compatible with Google Calendar, Outlook, Apple Calendar, and Samsung Calendar.  
It includes only the Romanian Orthodox red cross holidays for the year 2026.

### Import
You can download the file and import it into your preferred calendar application.
